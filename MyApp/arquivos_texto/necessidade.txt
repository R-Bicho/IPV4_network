Cada sub-rede tem um endereço para representá-la e essas sub-redes são interconectadas por meio de um roteador. O roteador precisa de um endereço de rede e de uma máscara de sub-rede para descobrir se o pacote de IP recebido deve ser roteado para uma de suas sub-redes. A realização de cálculos de rede pode ser difícil quando você tem que considerar parâmetros diferentes para saber quantas sub-redes devem existir e quantos hosts devem estar em uma sub-rede.

A calculadora de sub-redes é uma ferramenta útil para a descoberta do número de sub-redes possíveis para qualquer bloco de endereço de rede. Você pode escolher a combinação de sub-redes e número de hosts por sub-rede que é adequada para a sua rede e obter o intervalo de endereço do host e o endereço de transmissão para qualquer máscara de sub-rede fornecida. Particionar uma rede grande e alocar intervalos de endereços IP para equipes diferentes é uma tarefa que pode ser calculada mentalmente, mas é melhor ter uma opção como uma calculadora de intervalo de IP ou calculadora de máscara de sub-rede para verificar novamente seus cálculos de sub-rede antes de configurá-los no roteador.
